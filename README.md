# SQL na Prática

Série de estudos e posts sobre SQL aplicada ao contexto de análise de dados e negócios, com foco em exemplos simples, desafios e soluções comentadas.

## Objetivo
Transformar conceitos básicos de SQL em aplicações práticas para análise.

## Estrutura
Cada pasta representa um post/aula da série:
- conceito
- exemplo
- desafio
- solução comentada

## Série
- [Post 001 — SELECT](./post001-select/README.md)
- [Post 002 — WHERE](./post002-where/README.md)
- [Post 003 — DISTINCT](./post003-distinct/README.md)

## Como usar
1. Leia o README do tópico.
2. Rode o arquivo `vendas_schema.sql`.
3. Tente resolver o `desafio.sql`.
4. Compare com o `solucao.sql`.
